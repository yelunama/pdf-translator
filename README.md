# PDF专业翻译工具

一个专业级别的PDF文档翻译Web应用，支持英文到中文的高质量翻译，同时保留原始文档的格式和排版。

## 功能特点

- **文件上传**：支持拖放或选择PDF文件上传
- **专业翻译**：高质量的英文到中文翻译
- **格式保留**：翻译后保持原始PDF的格式和排版
- **对照预览**：支持原文/译文对照查看、仅原文查看、仅译文查看三种模式
- **文件下载**：可下载原始文件和翻译后的文件
- **翻译质量评估**：提供准确性、流畅度、术语一致性等评分
- **最近文件**：记录并快速访问最近处理的文件
- **分享功能**：生成分享链接，支持社交媒体分享
- **响应式设计**：适配桌面和移动设备
- **深色模式**：支持主题切换

## 技术栈

- HTML5
- CSS3 (Tailwind CSS v3)
- JavaScript
- PDF.js (Mozilla的PDF渲染库)
- GSAP (GreenSock动画平台)

## 本地运行

1. 克隆或下载项目到本地
2. 使用现代浏览器直接打开 `index.html` 文件
3. 无需安装其他依赖，所有外部库都通过CDN加载

## 注意事项

- 目前仅支持英文到中文的翻译
- 支持的PDF文件最大为20MB
- 本演示版本使用模拟数据，实际翻译功能需要连接到翻译API
- 所有文件处理在客户端完成，确保数据安全
